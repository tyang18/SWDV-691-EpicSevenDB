External Tools


YufineBot, A Discord Bot for E7

Created by Dimbreath, it's a Discord Bot that can be added to any Server, and it brings multiple E7 information as well as a Summon Simulator. Using EpicSevenDB API's data.

Equipment Preview/Planner Tool

Created by stocky37, it's a Character Equipment Preview/Planner, and as the name says, you can plan your current Hero, see ways of improving, all without spending Gold in-game swapping items around. Using EpicSevenDB API's data.

EpicSeven Tools

Created by budandan, it's a Damage Calculator, a Lab Team Optimizer and All Camping Teams DB. Using EpicSevenDB API's data.

E7DB DataBuilder

Created by Squachie, it's a Hero Data Builder, which is used to create new hero data for EpicSevenDB API. New hero being released and you want to contribute? Fill up the hero's details and send us the JSON it exports :)

Abyss Team Selector

Created by Wilson Yung & Julianne Lin, it's a Abyss Team builder, that users can create teams and share their strategies for each floor. Using EpicSevenDB API's data.

Discord Tier List

Created by Discord Community, this tier sheet is a compilation of the units strength and their paired artifacts.
